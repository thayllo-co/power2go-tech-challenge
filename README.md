## Executar Localmente

#### Requisito: [Node.js](https://nodejs.org/en/download/current)

Navegue até a pasta do projeto
~~~bash  
cd caminho/da/pasta/do/projeto
~~~

Instale as dependências
~~~bash  
npm install
~~~

Iniciar a aplicação localmente
~~~bash  
npm start
~~~
